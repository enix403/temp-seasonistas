GET /api/job/search
Allowed to: Everyone
Description: Allows the employee to search for jobs (using filters like job title, location, job type, etc.)

GET /api/job/:jobId
Allowed to: Everyone
Description: Returns detailed information about a specific job posting.

POST /api/job/:jobId/apply
Allowed to: Logged in employee
Description: Allows the employee to apply for a job.

GET /api/job/application/:applicationId
Allowed to: Logged in employee and employer associated with the application (applicant and job poster)
Description: View details of a specific application.

GET /api/employee/my-applications
Allowed to: Logged in employee
Description: Fetch a list of applications created by the employee.

-------------

POST /api/job/post
Allowed to: Logged in employer
Description: Allows an employer to post a new job

GET /api/employer/my-postings
Allowed to: Logged in employer
Description: Returns a list of jobs posted by the employer

GET /api/job/:jobId/applicants
Allowed to: Logged in employer associated with the job (job poster)
Description: Fetch a list of employees who have applied for a specific job posting.

PATCH /api/job/:jobId/update-status
Allowed to: Logged in employer associated with the job (job poster)
Description: Allow the employer to mark a job posting as active or inactive.

PATCH /api/job/application/:applicationId/update-decision
Allowed to: Logged in employer associated with the application (job poster)
Description: Allow the employer to update the decision of the application

PATCH /api/job/application/:applicationId/mark-interested
Allowed to: Logged in employer associated with the application (job poster)
Description: Allow the employer to mark if they are interested in the application.

GET /api/me/profile
Allowed to: Logged in user
Description: Get the current user's profile

PATCH /api/me/profile
Allowed to: Logged in user
Description: Update the current user's profile (complete or some fields)

GET /api/user/:userId
Allowed to: Everyone
Description: Get any user's profile

GET /api/admin/users
Allowed to: admin
Description: Fetches a list of all users (employees and employers) for administrative purposes.

PATCH /api/admin/user/:userId/ban
Allowed to: admin
Description: Allows the admin to ban a user from the platform.

-------------

DELETE /api/job/:jobId
Allowed to: admin
Description: Allows the admin to remove a job posting from the platform.

PATCH /api/admin/mark-approval
Allowed to: admin
Description: Allows the admin to approve or reject a user account after registration.

GET /api/admin/stats
Allowed to: admin
Description: Provides key metrics and statistics on platform usage (e.g., active users, jobs posted, applications submitted).
